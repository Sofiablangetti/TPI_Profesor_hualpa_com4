Descripción del programa:

Este programa permite cargar, visualizar y analizar datos de colegios a partir de un archivo CSV. Está diseñado para facilitar la exploración de información educativa, como provincia, nombre del colegio, cantidad de estudiantes y año de creación. Incluye funciones de búsqueda, filtros por provincia o por rango, ordenamientos y estadísticas generales. Además, valida errores comunes en la entrada del usuario (como acentos, espacios o datos faltantes) para mejorar la experiencia.

Instrucciones de uso:

Guardá el archivo CSV como colegios.csv en la misma carpeta que el programa.

Asegurate de que tenga columnas: provincia, colegio, estudiantes y anio_creacion (pueden tener acentos o mayúsculas).

Ejecutá el programa con:
python MenuPrincipal.py

Usá el menú para:
-Buscar colegios por nombre.
-Filtrar por provincia.
-Filtrar por rango de cantidad de estudiantes o año de creación.
-Ordenar colegios por nombre, cantidad de estudiantes o año de creación.
-Ver estadísticas generales.
-Agregar un colegio manualmente.
-Editar la cantidad de estudiantes y el año de creación de un colegio existente.
-Guardar los cambios en el archivo CSV.

Escribí el número de la opción y seguí las instrucciones.

Para salir, elegí la opción 10.

Ejemplo de entrada y salida:

Buscar por nombre:

Entrada: San Martín
Salida:
Se encontraron 1 colegio(s) con el nombre 'San Martín':
Instituto San Martín | Provincia: Córdoba | Estudiantes: 520 | Año: 1985
